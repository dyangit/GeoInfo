GeoInfo provides you the ability to click anywhere on the map to generate useful current information for that area.

Features

Map

  • On each each click, that is not on a marker, you should receive a popup notification showing the lat/long, current weather for that point, and geodata address.
  • Additionally for each click you make, based on the username, the query is stored in the Cloud through AWS services and is retrievable or deletable as you wish.

Both options have tooltip hover boxes to further give additional information as needed for each of the options they provide.

Known Issues

These are known issues with the web application that the developers know about and are fixing

  • Due to HTTP call lags, clicks on the map will typically be 'one-click-behind' with the popup message displaying prior click information. To fix this issue, simply double click on the intended area you wish to get information from.
  • The markers are not dynamically adjusted to a zoomed out map tile. To get around this issue, zoom back in to the initial state or click the 'Reset Zoom' button.
