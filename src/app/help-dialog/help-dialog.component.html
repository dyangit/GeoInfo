Overview

  • GeoInfo is application that allows the user to receive helpful data based on the region where they click on the map.
  • On each each click, the user should receive a popup notification showing the retrieved latitude and longitude, current data for that point, and geodata address.
  • Additionally for each click the user makes, based on the username, the query is stored in AWS services and is retrievable or deletable as the user wishes.
